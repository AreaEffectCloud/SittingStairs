# SittingStairs:chair:
### You can sit on the stairs in your own edited [:memo:resources/stairs.yml](https://github.com/AreaEffectCloud/SittingStairs/blob/master/src/main/resources/stairs.yml) file

## How to edit stairs.yml
### Example:
```
stairs:
  - ACACIA_STAIRS
  - ANDSITE_STAIRS
  - BIRCH_STAIRS
  - BLACKSTONE_STAIRS
  - BRICK_STAIRS
  - COBBLESTONE_STAIRS
```
